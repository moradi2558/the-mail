# D-Mail Project

پروژه سرویس ایمیل با Django و REST API

## ساختار پروژه

```
the-mail/
├── backend/          # پروژه Django (Backend API)
├── frontend/        # پروژه Frontend (آماده برای توسعه)
├── venv/            # محیط مجازی Python
└── README.md        # این فایل
```

## Backend

تمام فایل‌های Django در پوشه `backend/` قرار دارند.

برای راه‌اندازی:
```bash
cd backend
python manage.py runserver
```

برای اطلاعات بیشتر به `backend/README.md` مراجعه کنید.

## Frontend

پوشه `frontend/` برای توسعه رابط کاربری آماده است.

می‌توانید از فریمورک‌های مختلف استفاده کنید:
- React
- Vue.js
- Angular
- Next.js

برای اطلاعات بیشتر به `frontend/README.md` مراجعه کنید.
