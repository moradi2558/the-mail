# Frontend

این پوشه برای فایل‌های frontend پروژه است.

## توضیحات

پوشه frontend برای توسعه رابط کاربری (UI) استفاده می‌شود.

می‌توانید از فریمورک‌های مختلف استفاده کنید:
- React
- Vue.js
- Angular
- Next.js
- یا هر فریمورک دیگری که ترجیح می‌دهید

## راه‌اندازی

پس از انتخاب فریمورک، دستورالعمل‌های مربوط به آن را دنبال کنید.

## اتصال به Backend

Backend API در آدرس زیر در دسترس است:
```
http://localhost:8000/api/
```
